# tuui-vue
> A Vue.js 2.5 Component Library for Web

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```

### Deploy
```
yarn run deploy
```

## Solve some problems
```
yarn add -D vue-loader@14.2.2
```
